# ReadME for Project 8 - Neighbourhood Maps

This is the eighth and final project for the Udacity Front End Development Nanodegree.

# How to Install

Clone my GitHub Repo
git clone https://github.com/alicethorn/see-the-sites-of-cambridge.git

Using a terminal:
`npm install`
`npm start`

Navigate to http://localhost:3000/ in browser

# Dependencies
google-maps-React
escape-string-regexp

and only works in production mode.

All application components are usable across modern desktop, tablet, and phone browsers. Thes render in a responsive manner.
